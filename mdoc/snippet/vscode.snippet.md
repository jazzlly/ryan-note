# vscode snippet

### multi cursor
```
alt-click
ctrl-alt up/down:  multi cursor select

ctrl-d： 选中单词, ctrl-d, 选中下一个单词
ctrl-shift-l: 类似多个ctrl-d
```

### expand selection
```
shift-alt-left/right
```

### column select
```
shift-alt selection
```


### 列出文件中所有函数
```
ctrl-shift-o
idea: ctrl-f12
```


### switch between panel
```
ctrl 1, 2, 3
```

### fold/unfold
```
ctrl k, l: toggle fold
ctrl-shift-[, ]: fold/unfold
```
